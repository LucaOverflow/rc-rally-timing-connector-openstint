# rc-rally-timing-connector-openstint

```
Usage: rc-rally-timing-connector-openstint <decoder_key> [options]

This program connects an openstint decoder to a rc-rally-timing instance

Arguments:
  decoder_key              The Decoder Access Key taken from your rc-rally-timing dashboard

Options:
  --decoder <address>      The network address of the decoder (default: "localhost:5556")
  --rc_rally_timing <url>  The URL of the rc-rally-timing instance (default: "https://rc-rally-timing.com")
  -h, --help               display help for command
```
